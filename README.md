# MLOps_Full_ProjectCode
Storage used: Google Drive api and storage.\n
IDE used: Google Colab.\n
MLOps Project -\n This project is consisting of 3 sections -->  1.[TelemetryData]-Fetching Raw data from the api having telemetrydata records,2.[AnimalData]- (animal_id).csv is generated for each animalid from telemetrydata and processing and calculation are done to find the vedba_value,vedba_encoded_value (One-Hot_encoding) 3.[PredictedData] the latest animalid is used for making prediction using Multinomial Hidden Markov Model to predict the Hidden state of that animal id.
